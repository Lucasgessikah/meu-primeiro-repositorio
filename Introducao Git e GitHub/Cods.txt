git clone + link: Clona o repositório localmente na máquina.

git status: visualisa a situação atual da pasta.

git add . : adiciona todos os arquivos

git commit -m "alguma menssagem" : cria o commit para os devidos arquivos.

git push origin main: envia os arquivos locais para o servidor do GitHub.



git remote -v  (Ver qual o link esta apontado)

git remote rm origin (remover link)

git remote -v  (Confirmar que removeu o link)
